# trabalhoCompiladores
Construindo um analisador léxico 

Pra compilar no windows, abre o wsl e instala o bison com:

    sudo apt update && sudo apt install flex bison

Abre o wsl no diretório que os arquivos estão e roda esses comandos nele:

    bison -d parser.y
    flex trabalho.lex
    gcc -o compilador.flex.o -c lex.yy.c 
    gcc -o compilador.y.o -c parser.tab.c 
    gcc -o compilador compilador.flex.o compilador.y.o -lfl

Pra testar o arquivo de entrada "teste.txt" com o compilador criado:

    ./compilador teste.txt

Se tudo deu certo ele vai mostrar "Análise concluída com sucesso" no final
